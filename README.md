# Data Warehouse project from AWS S3 to Snowflake

Welcome to the Data Warehouse project repository that enables Data Ingestion from S3 to Snowflake, Data Transformation and preparing the gold layer (from Medallion architecture) for analytics-ready data. This project demonstrates a comprehensive Data Warehousing solution, from building a Data Warehouse to generating actionable insights. Designed as a portfolio project, it highlights industry best practices in Data Engineering.
